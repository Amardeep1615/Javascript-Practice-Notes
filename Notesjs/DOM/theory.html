DOM - Document Object Model Examples

1. DOM Selectors

1.1 getElementById

Hello by ID
document.getElementById("myId").style.color = "blue";

1.2 getElementsByClassName

Class Item 1
Class Item 2
let items = document.getElementsByClassName("myClass"); items[0].style.fontWeight = "bold";

1.3 getElementsByTagName

Paragraph 1

Paragraph 2

let tags = document.getElementsByTagName("p"); tags[1].style.color = "green";

1.4 querySelector

Box
document.querySelector(".box").style.border = "2px solid red";

1.5 querySelectorAll

  • Item A
  • Item B
document.querySelectorAll(".item").forEach(el => el.style.color = "purple");

1.6 addEventListener

document.getElementById("btn").addEventListener("click", () => { alert("Button clicked!"); });

2. DOM Manipulation

2.1 Create Element, AppendChild

let newElement = document.createElement("p"); newElement.textContent = "This is a new paragraph!"; document.getElementById("container").appendChild(newElement);

2.2 Remove Element

Remove this element
document.getElementById("removeMe").remove();

2.3 Replace Element

Old Element
let newDiv = document.createElement("div"); newDiv.textContent = "New Replaced Element"; let oldDiv = document.getElementById("replaceThis"); oldDiv.parentNode.replaceChild(newDiv, oldDiv);

3. Attribute Manipulation

3.1 getAttribute / setAttribute / hasAttribute

Visit
let link = document.getElementById("link"); console.log(link.getAttribute("href")); // https://example.com link.setAttribute("target", "_blank"); console.log(link.hasAttribute("target")); // true

4. Class Attribute Manipulation

Class Box
let box = document.getElementById("classBox"); // Add Class box.classList.add("newClass"); // Remove Class box.classList.remove("highlight"); // Toggle Class box.classList.toggle("toggleClass");
